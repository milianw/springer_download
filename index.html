springer_download by milianw

a python script for downloading ebooks from springerlink.com

Downloads all chapters of ebooks hosted on SpringerLink and merges them into one PDF file.

Dependencies

Python 2.4.x (2.6.x should work as well) pdftk iconv

Install

download and run: ./springer_download.py -l "http://springerlink.com/content/..."
